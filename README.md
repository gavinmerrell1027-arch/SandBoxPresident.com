# SandBoxPresident.com

A fantasy presidency simulator game built with React, TypeScript, and Tailwind CSS.

## Game Overview

Play as the President of the United States and navigate historical and fictional scenarios. Make tough decisions, manage advisors, balance budgets, and try to maintain approval ratings across different political parties.

## Features

- **Historical Presidents**: Play as Lincoln, Theodore Roosevelt, FDR, JFK, Reagan, and Obama
- **Dynamic Events**: Navigate political, economic, domestic, and environmental crises
- **Advisor System**: Hire advisors with different philosophies to boost various stats
- **Political Capital**: Manage your influence to pass executive orders and policies
- **Multiple Difficulty Levels**: Easy, Normal, Hard, and Impossible modes

## Game Stats

- **Approval**: Public support for your presidency
- **Economy**: Economic health and growth
- **Military**: Armed forces strength and readiness
- **Health**: Public health and healthcare system
- **Environment**: Environmental protection and sustainability
- **Debt**: National debt level

## Getting Started

```bash
npm install
npm run dev
```
